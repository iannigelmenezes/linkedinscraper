# LinkedIn Scraper

A LinkedIn feed scraper built with Playwright and Microsoft Edge support.

## Features
- Scrapes LinkedIn feed posts
- Uses Microsoft Edge browser
- Manual login for safety
- Extracts post content and metadata

## Setup
1. Install dependencies: `pip install playwright beautifulsoup4`
2. Run the scraper: `python linkedinscraper.py`

## Usage
The scraper will open Edge, prompt you to log into LinkedIn manually, then scrape your feed and display results in the terminal.
